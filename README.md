# SwiftUI Weather App (WeatherKit)

This sample SwiftUI app shows current weather for a curated list of American cities using WeatherKit.

## Features
- Current conditions powered by `WeatherKit`.
- City picker for major US cities.
- Loading and error states.

## Requirements
- Xcode 15+
- iOS 17+
- WeatherKit capability enabled for your app target.

## Notes
WeatherKit requires an Apple Developer account with the WeatherKit entitlement enabled. Add the capability in Xcode and run on a device or simulator signed with a valid team.
